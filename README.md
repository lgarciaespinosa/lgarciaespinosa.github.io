# 2048 (Uncluttered Version)

This is an "uncluttered" version of the original 2048 game, specifically adapted for people with low vision. 

### Key changes:
- **Responsive Fullscreen**: The board expands to fill the maximum available screen space.
- **Improved Visibility**: High-contrast, larger numbers inside tiles.
- **Minimalist Interface**: Removed all distractions, ads, and extra text.
- **Accessibility Feature**: Hold the **Enter** key for 2 seconds to restart the game.
- **PWA Support**: Works offline and can be installed on mobile devices.

Based on the original 2048 by Gabriele Cirulli: [https://github.com/gabrielecirulli/2048](https://github.com/gabrielecirulli/2048)
